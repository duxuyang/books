# mybooks

> 技术栈：vue + vue-router + vuex + webpack + axios  + flex + express + nodejs + mysql + localStorage

> [api地址](https://github.com/duxuyang/book-node)


## 使用说明

``` bash
#克隆项目
git clone https://github.com/duxuyang/books.git

# 安装依赖
npm install

# 本地开发环境 访问http://localhost:8080
npm run dev

# 构建生产
npm run build

```
